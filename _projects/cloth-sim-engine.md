# 柔性物体操作仿真引擎及系统搭建

**杰克科技 / 艾图机器人** · 2026.03 ~ 至今

## 项目概述

主导建设柔性物体操作数据飞轮基础设施，实现"真实采集 → 仿真重建 → 合成数据生成 → VLA评估与训练"闭环，为具身柔性操作提供高保真、可交互、标准化仿真系统底座。

## 技术栈

`Newton` `Isaac Sim` `3DGS` `ROS2` `SoftMimicGen` `Docker`

## 主要工作

1. **仿真引擎联调** — Isaac Sim + Newton 仿真引擎联调，攻克布料动态仿真与机械臂刚体物理耦合难点
2. **3DGS 点云重建** — 协调 3DGS 软硬件资源，实现场景 3D 高斯点云训练与 Isaac Sim 导入渲染
3. **Genie Sim 融合** — 利用 LLM 和 MCP 驱动柔性物体、本体、背景等资产构建仿真场景
4. **数据采集与生成** — Piper 真机 + ROS 数据采集，域随机化 + SoftMimicGen 合成数据生成
5. **VLA 推理管线** — 打通 VLA 推理，适配 RLinf 等多类算法，Docker 封装支持分布式训练

## 项目成果

推动工程化落地公司首套柔性物体操作的仿真引擎与完整系统，构建"实景采集 → 仿真重建 → 合成数据生成 → 算法训练 → 规模化评估"全链路数据飞轮闭环。
