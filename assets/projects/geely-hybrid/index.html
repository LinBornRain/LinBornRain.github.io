Overview

项目概述

以 WLTC 为目标工况，以控制混动模式切换和发动机输出功率为手段，以保证电平衡下累计油耗最低为目标，验证 Model-based Offline RL 技术在汽车混动标定问题上的有效性。

Approach

📊

特征工程

对客户提供的混动车辆运行数据进行清洗、分析和特征工程处理，构建训练数据集。

🔧

P1P3 串并联架构

WLTC 工况下较优的混动架构，优化空间为模式切换策略（纯电-串联-并联）与发动机输出策略。

🧠

环境动力学模型

构建环境模型：输入当前状态及混动模式，输出下一时刻状态与油耗奖赏反馈。

🎯

模式切换智能体

设计离散动作空间智能体：输入当前状态，输出纯电/串联/并联三种离散混动模式。

优化空间定义

  • 纯电模式 (EV) — 低速低负载，最大化电能利用率
  • 串联模式 (Series) — 中速中负载，发动机发电驱动电机
  • 并联模式 (Parallel) — 高速高负载，发动机直驱 + 电机辅助

Outcome

项目成果

核心成果

  • Offline RL 最优策略在真实环境中对比客户历史最佳策略，在相同条件下实现 油耗优化 0.9%
  • 验证 Model-based Offline RL 在汽车混动标定领域的技术可行性
  • 建立从数据对接、特征工程、环境建模到策略评估的完整研发流程

技术能力

  • 掌握 Model-based Offline RL在工业场景的落地全流程
  • 具备车辆动力学环境建模、离散动作空间策略设计能力
